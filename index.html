🎯 SINGLE PURPOSE

VideoLinkFinder has one purpose: detect and extract direct video source URLs from any webpage. It monitors network requests made by your browser to find video streams (HLS, DASH, MP4, WebM and other formats) and displays them in the extension popup so you can copy or open them.

All processing happens entirely within your browser. No data leaves your device.

🚫 DATA WE DO NOT COLLECT
✕ Personal information
✕ Browsing history
✕ URLs you visit
✕ Video links you copy
✕ IP address or location
✕ Cookies or identifiers
✕ Usage analytics
✕ Crash reports
💾 LOCAL STORAGE

The only data stored by this extension is your language preference (English, Russian or Ukrainian), saved locally on your device using chrome.storage.local. This setting never leaves your device and is never transmitted anywhere.

🔐 PERMISSIONS EXPLAINED
  • webRequest Monitors network requests to detect video file URLs (.m3u8, .mpd, .mp4, etc.) as they load. No request content or user data is read — only the URL pattern is checked.
  • activeTab Identifies the current tab to correctly associate detected video sources with the page you are viewing.
  • scripting Injects a content script to patch XMLHttpRequest and fetch on the page, catching video URLs requested by JavaScript-based video players.
  • storage Saves your language preference locally on your device only.
  • tabs Used to scope detected video sources per tab and to open video URLs in a new tab when you click "Open".
  • <all_urls> Required because video content can be served from any domain or CDN. The extension only inspects URL patterns — it does not read page content, form data or any personal information.
📡 THIRD PARTIES & REMOTE CODE

VideoLinkFinder does not use any third-party services, analytics platforms, advertising networks or remote code. All extension code is bundled locally within the extension package and reviewed by the Chrome Web Store.

✉️ CONTACT

If you have any questions about this privacy policy, please open an issue on the extension's GitHub repository or contact us via the Chrome Web Store support page.
